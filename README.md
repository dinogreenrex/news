# news app backend and frontend
```
Laravel 5.5;
1 cd news-api 
2 composer install 
3 mv env.txt to .env
4 run php artisan key:generate and paste the value in place of the existing 
 value in .env file under APP_KEY
5 provide database credentials in .env file
6 run php artisan migrate
7 run php artisan db:seed

-----
React: 
cd news-platform 
npm install 
npm start 

--- 
Copy the apache conf files into your sites-available dir, change the paths, 
 enable them all. 
```

Voila everything is up and running
